# Practice - Playwright tests

## Purpose
Examples of Playwright tests in the website https://practicesoftwaretesting.com/. 

## Installation
Open a console and run `npm i`

## Instructions
- To run the homepage tests: `npm run test:home`.
- To run the checkout tests: `npm run test:checkout`.
- To generate snapshots for the visual tests: `npm run test:visualGen`.
- To run the visual tests against the generated snapshots: `npm run test:visualNoGen`.
- To run the API tests: `npm run test:api`.

## Tools
Playwright / JavaScript
